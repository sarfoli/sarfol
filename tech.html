This part functions to tie all the components of the computer together and allows them to communicate with each other. Similar to the central nervous system, receiving peripheral signals, processing them, and sending them to an output source/device.

The CPU is the brain of the computer. It receives an input, stores data (much like the brain), and produces output results. This would make the CPU cooler of the CPU comparable to the human skull! It's a protective barrier to the brain of the system.

This is the computer's main memory. It's where the operating system (OS), data, and applications are stored for quick/temporary access. This is compared to the pre-frontal cortex of the brain, where short-term memory is stored/maintained.

These are permanent data storage devices, where data is stored and read. Similar to RAM, the HHD/SSD is responsible for memory and is compared to the brain. However, it is most similar to the hippocampus, which is responsible for permanent storage of memory.

This is the power supply, which functions like the mitochondrion (the powerhouse of the cell).
